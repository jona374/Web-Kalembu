# Imágenes de Kalembu

Aquí van las fotografías oficiales de la marca. Cada subcarpeta corresponde a una
sección de la web. Sube tus fotos con los **nombres exactos** de abajo y luego
yo las conecto al sitio (o avísame y lo hago).

## Recomendaciones generales

- Formato: **JPG** para fotos, **WebP** si quieres máxima velocidad, **PNG/SVG** para logos.
- Orientación vertical para producto y modelo (las prendas se ven mejor así).
- Peso: idealmente **menos de 400 KB** por imagen para que cargue rápido en el celular.
- Fondo limpio y luz uniforme para las fotos de producto.

## Dónde va cada imagen

### `hero/` — Portada principal
- `hero.jpg` — Foto editorial grande de una modelo con los jeans. **Horizontal**, mínimo 1920×1080 px.

### `coleccion/` — Los 3 colores
- `color-01.jpg`
- `color-02.jpg`
- `color-03.jpg`
- Una foto por color. **Vertical**, proporción 3:4 (ej. 900×1200 px).

### `producto/` — Producto destacado (las 4 miniaturas)
- `frente.jpg` — Vista frontal del jean.
- `espalda.jpg` — Vista posterior.
- `tela.jpg` — Detalle de la tela/costuras.
- `modelo.jpg` — El jean puesto en una modelo.
- **Vertical**, proporción 4:5 (ej. 1000×1250 px).

### `editorial/` — Bloque "Nuestra esencia"
- `editorial.jpg` — Foto de estilo editorial. **Vertical**, proporción 4:5.

### `galeria/` — Lookbook "Un jean, múltiples estilos"
- `look-01.jpg` (Minimal)
- `look-02.jpg` (Casual)
- `look-03.jpg` (Elegante)
- `look-04.jpg` (Auténtica)

### Logo (opcional, va en la raíz `img/`)
- `logo.svg` o `logo.png` — Si tienes el logo en imagen además del texto.
